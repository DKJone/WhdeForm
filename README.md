# WhdeForm
iOS 表格
```objective-c
pod 'WhdeForm', '~> 1.0.0'
```
- 添加了Reusable机制
- 添加了横（Section）表头
- 添加了竖（Column）表头
- 添加了左上角（TopLeftHeader）总表头
- 通过FDateSource去创建各个元素，类似TableView
- 添加了FIndexPath,{section, column}
- 表头添加点击事件
- 添加了网格
